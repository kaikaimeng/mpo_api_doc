class dso_sns.LineNotify(token: str = None)[source]

Bases: object

This can send Line Notify.

Important

Need to have your own token, https://notify-bot.line.me/en/

Important

Need configure your network DNS settings.

Useage:

import dso_sns
TOKEN = <your tiken>
l = dso_sns.LineNotify(TOKEN)
data = l.make_data('Message')
l.sned(data)
